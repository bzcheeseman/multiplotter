Plots the data! Takes a map (dictionary) where the key is the format and a number to distinguish the entries and the value is a tuple of vectors that represent x and y. Throws assertion error if x and y are not the same length.

Parameters
dataThe map of data values to plot
nameThe name to assign the plot if it's being saved.
sWhether to show the plot or not. If true, then the plot is not saved to a file. If false, the plot is saved to a file.
